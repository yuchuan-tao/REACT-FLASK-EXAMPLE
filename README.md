# This is an example of a Flask app with a React frontend. 

## Pathon Packages
```
pip install Flask
pip install flask_cors
pip install numpy
```

## Steps to run:
### Activate python virtual environment
```
cd flask-server
.\.venv\Scripts\activate
```
### Start the flask server
```
python server.py
```

### Start the react client
```
cd react-client
npm start
```
